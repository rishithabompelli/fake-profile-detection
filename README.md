# Fake Profile Detection System
 
A Machine Learning and Deep Learning system to detect
fake/bot profiles on social media platforms.
 
## Tech Stack
- Python, Flask
- TensorFlow, Keras (RNN/LSTM)
- Scikit-learn (Random Forest, SVM)
- NLP: TF-IDF, Tokenization, Sentiment Analysis
- Pandas, NumPy, Matplotlib
- MySQL
 
## Results
- LSTM Model Accuracy: 92%
- Precision: 94% | Recall: 91%
- Outperforms baseline by 35%
 
## How to Run
```
git clone https://github.com/rishithabompelli/fake-profile-detection
cd fake-profile-detection
python -m venv venv
venv\Scripts\activate  (Windows)
pip install -r requirements.txt
python model.py
python app.py
```
Open http://127.0.0.1:5000 in your browser.
